# Pengfei doc center


## How to update docs

To add more docs to the doc center, you need to follow the below steps:
1. Create a source file(e.g. intsall_nginx_in_debian.md) and place it in the right folder.
2. Add the source file to the `nav` section in the `mkdocs.yml`
3. Preview the new version of the doc center with `mkdocs serve` command 
4. Publish the new version via GitHub pages with `mkdocs gh-deploy` command
